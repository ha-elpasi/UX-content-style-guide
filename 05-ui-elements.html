UI Elements

Every piece of content we publish is supported by a number of smaller pieces. This section lays out our style for UI elements.

Guidelines

Alt text

Alt text is a way to label images, and it's especially important for people who can’t see the images in our interface. Alt text should describe the image in a brief sentence or two.

Buttons

Button copy should always include verbs. Keep things clear and concise, and use title case.

Checkboxes

Use sentence case for checkboxes.

Drop-down menus

Use title case for menu names and sentence case for menu items.

Links

Use descriptive link text. Don’t use “Click here” or “Read this.” Link relevant keywords instead.

Don’t include preceding articles (a, an, the) in the linked text.

Lists

Number lists when order matters. Use bullets when it doesn’t. Always introduce a list with a brief sentence. Capitalize the first word of each item.

Navigation

Use title case for main navigation. Use sentence case for subnavigation. Keep labels short.

Radio Buttons

Use title case for headings and sentence case for button fields.

Titles

Titles appear at the beginning of a page or section. They tell users what to expect.

Titles are written in title case. Don’t use end punctuation unless the title is a question.

The verb in the title must be repeated in the CTA button.
