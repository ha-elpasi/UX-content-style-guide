Word List

Standardized spellings

These words can be slippery. Here’s how we write them. (If it’s not on this list, defer to the AP Style Guide.)

  • add-on (noun, adjective), add on (verb)
  • checkbox
  • click (desktop interactions), tap (mobile interactions)
  • coworker
  • double-click
  • drop-down (noun, adjective), drop down (verb)
  • e-mail (always hyphenate, never capitalize unless it begins a sentence)
  • emoji (singular and plural)
  • error (not “mistake” or “problem”)
  • geolocation
  • homepage
  • internet (never capitalize unless it begins a sentence)
  • loading (use sparingly; only when the wait exceeds 3 seconds)
  • login (noun, adjective), log in (verb)
  • notification
  • OK
  • online (never capitalize unless it begins a sentence)
  • opt-in (noun, adjective), opt in (verb)
  • password
  • pop-up (noun, adjective), pop up (verb)
  • sign in (noun, adjective), sign in (verb)
  • signup (noun, adjective), sign up (verb)
  • sync
  • third party (noun), third-party (adjective)
  • username
  • URL
  • website
  • WiFi
  • ZIP code (always capitalize ZIP)
