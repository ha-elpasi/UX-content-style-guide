# Content Style Guide (Test Set)
 
A sample content style guide in multiple formats, created to test how format affects the performance of style guides used as knowledge sources for custom GPTs and AI agents.
 
The content is adapted from [Mailchimp's Content Style Guide](https://styleguide.mailchimp.com/), with some rules modified and all Mailchimp references removed.
 
## Formats
 
- Markdown
- HTML
- Word (single file)

## Sections
 
1. Writing Goals and Principles
2. Voice and Tone
3. Writing About People
4. Grammar and Mechanics
5. UI Elements
6. Word List


## Why
 
When building writing assistants, the same style guide can produce different results depending on whether it's uploaded as Markdown, HTML, or Word. Public style guides can't isolate what's coming from RAG vs. training data. So, I created a new, brand-agnostic one.
 
## Note
 
This is a test artifact, not an official style guide.
